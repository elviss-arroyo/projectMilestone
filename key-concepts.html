HTTP (Hypertext Transfer Protocol)

HTTP is the protocol that allows your browser to request information from a server and display web pages. It is the foundation of how the web works.

Tim Berners-Lee

Tim Berners-Lee invented the World Wide Web and created the first version of HTTP.

HTTP/3

HTTP/3 is the latest version of HTTP, making websites faster and more reliable.

HTTPS

HTTPS is the secure version of HTTP. It encrypts messages between your browser and the server, protecting sensitive information.

POST

POST is a type of HTTP request used to send data to a server.

GET

GET is a type of HTTP request used to retrieve data from a server.

DELETE

DELETE is a type of HTTP request used to remove data from a server.

404 Not Found

404 Not Found is a status code returned by the server when a requested webpage cannot be found.
