When you visit a website, here’s what happens step by step:

  1. You type the web address (like www.google.com) into your browser.
  2. Your browser sends a request to the server that hosts the website. This is like saying, "Hey, can I see this page?"
  3. The server gets the request, finds the page, and sends it back.
  4. Your browser shows the page on your screen.

HTTP uses a client-server model. Requests can be different types:

  • GET: Asking for something, like a webpage or image.
  • POST: Sending information, like filling out a form.
  • PUT: Updating something.
  • DELETE: Removing something.
  • HEAD: Same as GET but without the response body.
  • PATCH: Used to give modifications to a reseource.
